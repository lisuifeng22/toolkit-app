# 隐私说明

## 权限使用

本 App 可能请求以下系统权限：

| 权限                                                     | 用途           | 说明                                |
| -------------------------------------------------------- | -------------- | ----------------------------------- |
| 定位权限 (ACCESS_FINE_LOCATION / ACCESS_COARSE_LOCATION) | 天气功能       | 获取当前位置以提供当地天气信息      |
| 网络权限 (INTERNET)                                      | 天气、地理编码 | 请求高德地图 API 获取天气和地理数据 |

### 定位权限说明

- 定位权限**仅用于天气功能**的自动定位
- 如果用户拒绝定位权限，App 不会反复请求
- 拒绝定位后，App 会自动使用 **IP 定位** 或 **手动搜索城市** 作为替代方案
- 定位成功后，位置信息不会上传到除高德地图 API 以外的任何服务器

## 第三方服务

### 高德地图 API

- 天气数据和地理编码功能通过高德地图 Web 服务 API 实现
- 请求可能包含：
  - 城市名称或 adcode（手动搜索时）
  - 经纬度坐标（GPS 定位时）
  - IP 地址（IP 定位兜底时）
- 请参阅[高德地图隐私政策](https://lbs.amap.com/home/privacy/)了解其数据处理方式
- 本 App 不控制第三方服务对数据的使用

## 本地数据

| 数据类型     | 存储方式                         | 加密                      |
| ------------ | -------------------------------- | ------------------------- |
| 便签         | AsyncStorage (本地)              | 否                        |
| 待办         | AsyncStorage (本地)              | 否                        |
| 生日         | AsyncStorage (本地)              | 否                        |
| 纪念日       | AsyncStorage (本地)              | 否                        |
| 倒计时       | AsyncStorage (本地)              | 否                        |
| 天气缓存     | AsyncStorage (本地)              | 否                        |
| 密码本       | AsyncStorage (本地)              | AES-256-CTR + HMAC-SHA256 |
| PIN 密钥材料 | expo-secure-store (系统安全存储) | 系统级加密                |

- **所有数据均仅保存在设备本地**
- **App 不提供云同步功能**
- **卸载 App 将导致所有本地数据丢失**
- 密码本数据加密后存储，但本地加密不能替代专业密码管理器
- Root、越狱、恶意软件环境下仍可能存在安全风险

## 数据上传

- 本 App **没有后端服务器**
- App **不会主动收集、上传或分享** 用户数据
- 唯一的外部请求是高德地图 API（天气和地理编码），详见"第三方服务"部分

## 安全边界

- 本 App 的本地加密方案旨在保护日常使用中的数据安全
- 不建议使用此 App 存储极高敏感度的密码信息
- 在已 Root 或越狱的设备上，本地加密的保护效果会显著降低
- 请勿将 API Key 视为绝对机密，客户端 Key 可能被反编译获取
- 如需真正的端到端加密和专业密码管理，请使用专业密码管理器
